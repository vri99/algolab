The project covers different sorting and searching algorithms for several purposes:
  1. Learn Python
  2. Learn Algorithms
  3. Learn Data Structures

Searching algos:
  1. Binary search
  2. Prefix sum
  3. Depth-first search
  4. Breadth-first search
  5. Topological sort
  6. Dijkstra's search

Sorting algos:
  1. Quick sort
  2. Selection sort
  3. Merge sort

Data structures:
  1. Linked List
  2. Hash Table
  3. Binary Tree
